This file provides detailed information on the Arsenal 
font Software.

This information should be distributed along with the 
Arsenal fonts and any derivative works.

Basic Font Information

http://www.ukrainian-type.com/home/

Arsenal is a Unicode typeface family that supports languages that use the Latin, and Cyrillic scripts and its variants, and could be expanded to support other scripts.

More specifically, this release supports the following Unicode
ranges: Latin-1, GF Cyrillic Plus

To contribute to the project contact Alexei Vanyashin at a@cyreal.org

There are two source files
1. sources/Arsenal.glyphs
2. sources/Arsenal-Italic.glyphs
